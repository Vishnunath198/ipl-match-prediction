# 🏏 IPL Match Prediction

This project predicts IPL match outcomes using Machine Learning.

## 📊 Overview

The model analyzes historical IPL data and predicts match results based on:

* Teams
* Venue
* Toss decision
* Runs and wickets

## 🛠️ Tech Stack

* Python
* Pandas
* NumPy
* Scikit-learn
* Jupyter Notebook

## 🚀 How to Run

pip install -r requirements.txt
jupyter notebook

## 📈 Output

* Match winner prediction
* Model accuracy
